## Quick note:

The setup is not perfect and may experience occasional delays or minor bugs.

## Setup:
  1. The hand.py handles the tracking(CLIENT)
  2. The app.py handles the server(SERVER)
  3. RUN both on separate machines.
  4. Give hand.py the IPv4 local address of the server.
  5. Press the PERMISSION button once. This allows the browser to play the MP4 file without requiring additional user interaction.
  6. When the tip of the middle finger and the thumb comes near contact the script will run the script.JS, the script can be modified to the user's liking.
